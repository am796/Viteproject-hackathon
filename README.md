# Vite Dashboard Prototype (minimal)

This is a minimal prototype demonstrating a login-protected dashboard with summary cards, a Chart.js chart, and a details table.

Seeded credentials

- viewer@vite.co.in / pass123 (viewer role)
- analyst@vite.co.in / pass123 (analyst role)
- admin@vite.co.in / pass123 (administrator role - access to product dashboard)

Currency: amounts are displayed in Indian Rupees (₹) for this demo.

How to run (Windows PowerShell):

1. Install dependencies
   npm install

2. Start server
   npm start

3. Open the app
   Visit http://localhost:4000 in your browser.

Notes:
- This is a demo. Secrets are hard-coded and stored in memory. Do not use in production.
- The analyst role reveals a filter control and can see category filtering.
- The admin role can access the product dashboard and product API.
- The app simulates live updates every 15 seconds.
